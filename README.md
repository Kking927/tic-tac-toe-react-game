# Tic-Tac-Toe React Game

A simple Tic-Tac-Toe game built with **React**, **JavaScript**, and **CSS**, featuring 2-player gameplay, win/draw detection, and a reset button.

## Features

- 3x3 interactive game grid
- Player turns: X and O
- Win and draw detection
- Reset button to start a new game

## Built With

- React
- JavaScript
- Vite
- Vercel (deployment)
