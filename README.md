# Tic-Tac-Toe React Game

A simple **Tic-Tac-Toe game** built with **React, JavaScript, and CSS**.  
The game features 2-player gameplay, win/draw detection, and a reset button.

---

## Live Demo

[Tic-Tac-Toe](https://tic-tac-toe-react-game-zeta.vercel.app/)

---

## Features

- 3x3 interactive game grid
- Player turns: X and O  
- Win and draw detection 
- Reset button to start a new game  
- Clean and responsive UI

---

## Tech Stack Used

- **React** – Front-end library  
- **JavaScript** – Modern syntax  
- **Vite** – Build tool  
- **CSS** – Styling  
- **Vercel** – Deployment platform

---

## Skills Showcased

- Building interactive **2-player game logic**  
- Managing **state and player turns** with `useState`  
- Implementing **conditional rendering** for win/draw messages  
- Creating **responsive, styled UI** with CSS  
- Structuring a **small React project** for maintainability  
- Deploying a React app to **Vercel**
